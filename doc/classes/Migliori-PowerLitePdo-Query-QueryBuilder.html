$debugOnceMode

QueryBuilder.php : 74

Debug mode for the query builder.

private bool|string $debugOnceMode = false

debugGlobalMode is the global debug mode for the class. debugOnceMode is the debug mode for a single query.

If debugOnceMode is false, the global debug mode will be used. If debugOnceMode is set, it will override the global debug mode. Both can be set to false, true, or 'silent'.

If set to true, the class will display detailed error messages. If set to 'silent', the class will register all error messages, which can be retrieved using the getDebug() method.
